# Nu IA & Capability Map (v1)

## Philosophy
Nu is organized by **Capabilities**, not features. The interface is a portal to what the user can *do*.

## 1. Top Bar (Global Actions)
| Capability | Label | View | Purpose |
| :--- | :--- | :--- | :--- |
| **create** | Create Task | `CreateSheet` | Primary entry point for rapid action |
| **graph** | Task Graph | `GraphView` | Visualize dependencies and clusters |
| **search** | Search | `SearchView` | Global index retrieval |
| **settings** | Settings | `SettingsView` | App-wide configuration |
| **about** | About Us | `AboutView` | Platform context |

## 2. Bottom Bar (Capability Buckets)
Simplified to 5 core buckets that house modular focus areas.

### A. Today (`TodayView`)
Mental focus: *What is happening now?*
- **Todo**: The immediate backlog.
- **Focus**: The single active priority.
- **Sprint**: The current 90min/daily cycle.
- **Bingo**: Habit-tracking game loop.

### B. Assist (`AssistHub`)
Mental focus: *Help me process/think.*
- **LF Assists**: Domain-specific AI guidance.
- **Rant mode**: High-bandwidth voice/text ingest.
- **Chat**: Threaded AI interaction.
- **Tools**: Deterministic solvers (calculators, schedulers).
- **Solve**: The pipeline from raw input to structured action.

### C. Create (`+`) (`CreateSheet`)
Mental focus: *Generate something new.*
- **Task**: Standard actionable item.
- **Activity**: Narrative episode/moment.
- **Story**: Generative text reflection.
- **Comic**: Visual narrative render.
- **Report**: Structured accountability output.

### D. Me (`MeHub`)
Mental focus: *My identity and privacy.*
- **LLM Keys**: BYOK (OpenAI, Gemini, Anthropic).
- **Life Focus**: Core priority balancing (Core, Self, Impact, etc.).
- **Hidden LF Groups**: Privacy-filtered focus areas.
- **No Data AI**: Toggle for E2EE/Local-only AI processing.
- **Privacy**: Global data controls.

### E. Platform (`PlatformHub`)
Mental focus: *System meta-layer.*
- **Analytics**: DAU, Usage, Health.
- **Feedback**: Bug reports, feature requests.
- **Marketplace**: Store, Buy/Sell, Third-party integrations.
- **Enterprise**: Companies, Multi-tenant orgs.

## 3. Extended Capabilities (Route-Only)
These are accessible via deep links or hub navigation but do not occupy primary bottom-bar space.
- **Social**: Walls, Spaces, Feeds, Partner Linking.
- **Games**: Board views, Streak rings, XP decay.
- **Learn**: Library, Courses, Skill paths.
- **Business**: Commerce, Inventory, Storefronts.
- **Audits**: 7-Segment accountability segments.

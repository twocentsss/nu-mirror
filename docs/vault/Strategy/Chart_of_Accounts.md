# Chart of Accounts (CoA) v1.0

> **S F SS** (Series - Focus - Subfocus)
> A 3-part code system that won't explode.

## Code Format: `TYYY.FF.SS`
*   `TYYY`: Account Type (Series)
    *   `1xxx`: Assets (Capital built)
    *   `2xxx`: Liabilities (Debts/Backlog)
    *   `3xxx`: Equity (Values/Identity)
    *   `4xxx`: Returns (Value produced)
    *   `5xxx`: COGS (Direct doing)
    *   `6xxx`: Expenses (Maintenance/Leakage)
*   `FF`: Focus (The 9 Pillars, 01-09)
*   `SS`: Sub-focus (Leaf detail, 01-99)
